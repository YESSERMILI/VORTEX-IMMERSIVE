# **App Name**: Vortex Immersive

## Core Features:

- Dynamic Hero Canvas: Interactive Three.js based particle system with flowing streams and a central wireframe sphere, creating a captivating visual background for the hero section.
- Comprehensive Content Display: Structure and render diverse content across 10+ sections including 'Works', 'Services', 'About Us', and 'Contact', making use of various UI patterns like grids and cards.
- Works Filtering and Modals: Implement client-side filtering for 'Works' projects based on categories, and a robust modal system for displaying detailed project information or showreel videos.
- Global Theme Toggling: Allow users to switch between dark and light themes seamlessly, updating all relevant UI elements and ensuring accessibility.
- Advanced Interaction & Animation: Integrate custom cursor effects, preloader sequence, scroll-triggered animations (fade-in, slide-up, counter animations), and various hover effects for an engaging user experience.
- AI-Powered Project Inquiry Tool: In the contact form, provide an AI tool that assists users in drafting their project inquiries by suggesting relevant keywords, questions, or potential service integrations based on their input.

## Style Guidelines:

- Dark scheme, building upon existing neutrals. Primary accent color: Bright cyan-blue (#1ACCED, HSL H=200 S=90% L=50%) for a sophisticated and modern tech feel. Background color: A very dark, almost black, subtle blue-grey tint (#121617, HSL H=200 S=15% L=8%) derived from the primary, for depth. Accent color: A vibrant mint green-cyan (#66F0DB, HSL H=170 S=80% L=60%) to provide strong contrast and visual interest.
- Headlines and display text: 'Syne' (sans-serif) for its strong, modern, and condensed appearance, used for emphasis. Body text and general content: 'DM Sans' (sans-serif) for high readability and a clean, contemporary feel. Note: currently only Google Fonts are supported.
- Utilize a consistent set of minimalist, stroke-based line icons (e.g., from Feather Icons style) for UI elements, navigation, service highlights, and social links to maintain a sleek, high-tech aesthetic.
- Employ a modern grid-based layout for showcasing content sections (stats, services, works, etc.), ensuring responsive adaptation across devices. Content is centered and constrained within a maximum width of 1400px with generous horizontal padding, creating a premium feel. Full-height sections (e.g., hero) are used to emphasize key content.
- Incorporate sophisticated animations such as a dynamic preloader sequence, scroll-triggered fades and slide-ins for elements (.reveal class), subtle parallax effects on hero visuals, engaging hover states for interactive components, and numerical count-up animations for statistics sections, ensuring a fluid and premium user experience.
